# geekhubs.com

#### Install:

```
git clone https://github.com/kulakowka/geekhubs.com.git
cd geekhubs.com
npm install 
```

#### Development:

```
npm start
```

Open in your browser [http://localhost:3000](http://localhost:3000)

#### Production:

```
PORT=3000 NODE_ENV=production node ./bin/www --harmony
```
